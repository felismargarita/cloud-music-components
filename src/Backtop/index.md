## Backtop

### 何时使用:
在歌曲播放页面,滑动查看热评和最新评论时,下滑出现backtop组件,点击后滚动至页面最上方

### 代码演示:
```tsx
import React from 'react';
import { BackTop } from 'cloud-music-components';

export default ()=><div>
<span>向下滚动屏幕,显示backtop组件</span>
<BackTop/>
</div>

```
